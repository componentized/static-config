## Build

Prereqs:
- a rust toolchain with a recent nightly (`rustup toolchain install nightly`)
- [`cargo component`](https://github.com/bytecodealliance/cargo-component)
- [`wkg`](https://github.com/bytecodealliance/wasm-pkg-tools)

```sh
./build.sh
```
